# Список тестов

## Класс Lexer

### Базовые токены
- [ ] Пустой ввод: `""`
- [ ] Точка с запятой: `";"`
- [ ] Пробелы и табуляции: `"   \t  "`

### Ключевые слова C#
- [ ] `class`
- [ ] `if`
- [ ] `while`
- [ ] `return`

### Идентификаторы
- [ ] Простой идентификатор: `"variableName"`

### Числовые литералы
- [ ] Целое число: `"123"`

### Строковые литералы
- [ ] Простая строка: `"\"hello\""`

### Операторы
- [ ] Присваивание: `"="`
- [ ] Арифметические: `"+", "-"`

### Разделители
- [ ] Фигурные скобки: `"{", "}"`
- [ ] Круглые скобки: `"(", ")"`
